BrAPI Test Server

Welcome to the BrAPI example test server. This is a fully implemented example of the BrAPI specification, backed by a sample database. The server is written in Java using the Spring Boot library, the backing database is PostgreSQL. Take a look at the source HERE.

Here are some examples to get you started.

  • /brapi/v1/calls
  • /brapi/v1/crops
  • /brapi/v1/seasons?year=2013
  • /brapi/v1/studies/1001
